Lenguajes de Bajo Nivel

Assembly

Creado por: Varios autores

Año de creación: 1949

Utilidad: Se utiliza para programación de bajo nivel y manipulación directa del hardware.

Cualidades: Permite un control total sobre el hardware, pero es complejo y difícil de aprender.

C

Creado por: Dennis Ritchie

Año de creación: 1972

Utilidad: Es uno de los lenguajes de programación más importantes y se utiliza ampliamente en sistemas operativos y software de alto rendimiento.

Cualidades: Potente, eficiente y flexible; se usa para desarrollar sistemas operativos y aplicaciones de software que requieren un alto rendimiento.

C++

Creado por: Bjarne Stroustrup

Año de creación: 1983

Utilidad: Ampliamente usado en software de sistemas, aplicaciones de alto rendimiento, videojuegos y programación orientada a objetos.

Cualidades: Extiende a C con características de programación orientada a objetos; muy potente y usado en grandes proyectos y aplicaciones donde el rendimiento es crítico.

Fortran

Creado por: John Backus

Año de creación: 1957

Utilidad: Se usa principalmente en la programación científica y en áreas de cálculo numérico.

Cualidades: Es uno de los lenguajes más antiguos en uso y sigue siendo popular en las ciencias e ingeniería debido a su eficiencia en operaciones matemáticas complejas.

Pascal

Creado por: Niklaus Wirth

Año de creación: 1970

Utilidad: Se usa principalmente en enseñanza de programación, aunque también es útil en aplicaciones de software y sistemas pequeños.

Cualidades: Es un lenguaje estructurado, fácil de aprender y con una sintaxis clara. A menudo se utiliza como primer lenguaje de programación.

Lenguajes de Alto Nivel

Python

Creado por: Guido van Rossum

Año de creación: 1991

Utilidad: Python es un lenguaje versátil utilizado en ciencia de datos, desarrollo web, automatización y más.

Cualidades: Sencillo de aprender y leer, con una sintaxis clara y extensa librería estándar, popular en inteligencia artificial y desarrollo web.

Java

Creado por: James Gosling

Año de creación: 1995

Utilidad: Se usa principalmente en aplicaciones empresariales, sistemas distribuidos, y aplicaciones móviles (Android).

Cualidades: Lenguaje orientado a objetos, robusto y con una amplia comunidad de desarrollo; famoso por su principio "escribir una vez, correr en cualquier lugar".

JavaScript

Creado por: Brendan Eich

Año de creación: 1995

Utilidad: Usado principalmente para desarrollo web, creando interactividad en páginas web.

Cualidades: Ligero y rápido, soportado por todos los navegadores web, permite la creación de aplicaciones web dinámicas.

C#

Creado por: Anders Hejlsberg

Año de creación: 2000

Utilidad: C# es utilizado principalmente en el desarrollo de aplicaciones Windows, videojuegos (con Unity) y aplicaciones web.

Cualidades: Es un lenguaje orientado a objetos, muy moderno y con soporte para programación orientada a eventos y desarrollo de aplicaciones multiplataforma.

Lenguajes Funcionales

Lisp

Creado por: John McCarthy

Año de creación: 1958

Utilidad: Usado principalmente en inteligencia artificial, especialmente en procesamiento de lenguaje natural y en la creación de aplicaciones autónomas.

Cualidades: Su sintaxis es altamente flexible y sus macros permiten la creación de nuevos lenguajes de programación. Es conocido por su capacidad para manejar operaciones con listas.

Haskell

Creado por: Varios autores (comité de Haskell)

Año de creación: 1990

Utilidad: Usado principalmente en investigaciones académicas y aplicaciones que requieren funciones matemáticas precisas y manipulación de datos complejos.

Cualidades: Lenguaje puramente funcional, con énfasis en la corrección y la claridad del código.

Erlang

Creado por: Joe Armstrong

Año de creación: 1986

Utilidad: Usado principalmente en sistemas de telecomunicaciones y aplicaciones que requieren alta disponibilidad y procesamiento concurrente.

Cualidades: Su capacidad para manejar concurrencia masiva y recuperación ante fallos lo hace ideal para sistemas en tiempo real.

Lenguajes de Consulta y Bases de Datos

SQL

Creado por: Donald D. Chamberlin y Raymond F. Boyce

Año de creación: 1974

Utilidad: Usado para gestionar bases de datos relacionales, permitiendo la consulta, actualización y administración de datos.

Cualidades: Lenguaje declarativo fácil de aprender, que se utiliza en casi todos los sistemas de bases de datos relacionales.

GraphQL

Creado por: Facebook

Año de creación: 2012

Utilidad: Usado para hacer consultas y manipulación de datos en aplicaciones web modernas. Permite una mejor eficiencia y flexibilidad que REST.

Cualidades: Permite al cliente especificar qué datos necesita, lo que optimiza las consultas y reduce el sobrecosto de red.

SPARQL

Creado por: W3C (World Wide Web Consortium)

Año de creación: 2008

Utilidad: Se usa para realizar consultas sobre bases de datos RDF (Resource Description Framework), comúnmente usadas en la web semántica.

Cualidades: Permite la consulta y manipulación de datos en formato RDF, ampliamente utilizado para consultas de datos interconectados.

Lenguajes de Scripting

Bash

Creado por: Brian Fox

Año de creación: 1987

Utilidad: Usado para escribir scripts en sistemas UNIX y Linux, y automatizar tareas administrativas del sistema.

Cualidades: Sencillo de usar, con una gran capacidad de automatización y manipulación de archivos de texto.

PowerShell

Creado por: Microsoft

Año de creación: 2006

Utilidad: Usado para la automatización de tareas de administración de sistemas y para trabajar con la línea de comandos en Windows.

Cualidades: Extiende la línea de comandos de Windows con un enfoque en la administración del sistema y la manipulación de objetos.

Tcl

Creado por: John Ousterhout

Año de creación: 1988

Utilidad: Usado en aplicaciones que requieren scripts de automatización, integración de software y pruebas de sistemas.

Cualidades: Es fácil de integrar con otras aplicaciones y es muy usado en el desarrollo de sistemas embebidos y automatización.

Lenguajes de Markup

HTML

Creado por: Tim Berners-Lee

Año de creación: 1991

Utilidad: Es el lenguaje estándar para crear y estructurar páginas web en la World Wide Web.

Cualidades: Utiliza etiquetas para estructurar contenido; es el lenguaje esencial para la creación de sitios web.

XML

Creado por: World Wide Web Consortium (W3C)

Año de creación: 1998

Utilidad: Utilizado para el intercambio de datos entre diferentes sistemas y plataformas.

Cualidades: Permite la creación de documentos de datos estructurados con una sintaxis legible por máquinas y humanos.

YAML

Creado por: Clark Evans

Año de creación: 2001

Utilidad: Usado principalmente para configuración de aplicaciones y serialización de datos, especialmente en DevOps.

Cualidades: Su sintaxis es más simple y fácil de leer que JSON, lo que lo hace popular para la configuración de archivos.

Lenguajes para Desarrollo Web

CSS

Creado por: Håkon Wium Lie y Bert Bos

Año de creación: 1996

Utilidad: Usado para dar estilo y diseño a las páginas web creadas con HTML.

Cualidades: Permite separar el contenido de la presentación, lo que facilita la mantenibilidad y el diseño adaptable a diferentes dispositivos.

JavaScript

Creado por: Brendan Eich

Año de creación: 1995

Utilidad: Usado principalmente para hacer que las páginas web sean interactivas y dinámicas, siendo esencial para el desarrollo web moderno.

Cualidades: Es un lenguaje muy flexible que puede ser usado en el lado del cliente (navegador) y del servidor (Node.js).

Lenguajes para Blockchain y Contratos Inteligentes

Solidity

Creado por: Gavin Wood

Año de creación: 2014

Utilidad: Usado para desarrollar contratos inteligentes en la blockchain de Ethereum.

Cualidades: Específicamente diseñado para la creación de aplicaciones descentralizadas y contratos en blockchain, enfocado en seguridad y eficiencia.

Vyper

Creado por: Varios colaboradores

Año de creación: 2017

Utilidad: Un lenguaje alternativo a Solidity para contratos inteligentes, diseñado para ser más simple y más seguro.

Cualidades: Más restrictivo que Solidity, con menos funciones para evitar errores en contratos complejos.
